## 1. KEYBOARD HOTKEYS

Use these hardware overrides to navigate panels and modes:

| Key Binding | Command Trigger | Action Performed |
| :--- | :--- | :--- |
| `Ctrl + [` | `TOGGLE_SIDEBAR` | Collapses or expands the left configuration drawer. |

## 3. SURVEILLANCE CAMERA (CCTV) DECK

Clicking on a CCTV sensor node open-circuits a live surveillance feed overlay with simulated data feedback:
1.  **Select Camera**: Find cameras situated near Washington, New York, London, Tokyo, or Austin.
2.  **Inspect Feeds**: Clicking a camera opens a central popup window displaying system ID, name, status (`Recording`, `Online`, `Maintenance`), and an animated analog static scan overlay.
3.  **Terminate Connection**: Click the `[X]` button or hit `Escape` to shut the sensor feed down.
